# Conversor de moneda española del s. XVIII

Sencilla calculadora que permite encontrar las equivalencias entre las diferentes unidades de moneda española usadas en el siglo XVIII. 

## Monedas disponibles
- Maravedises
- Reales
- Ducados
- Reales de a 8
- Escudos
- Doblones de a 8

## Requisitos
- Python 3
- PyQt5
- NumPy
